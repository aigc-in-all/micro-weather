### 一个简单的天气预报微服务项目

本项目分为四个微服务实现：
1. 城市列表微服务
2. 天气数据采集微服务
3. 天气数据API微服务
4. 前端展示服务

涉及到的知识点有：
* Spring Boot
* String Cloud
* 服务注册与发现eureka
* 服务消费feign
* api网关zuul
* 中心化配置spring cloud
* 熔断hystrix
* Redis
* Docker
* quartz
* thymeleaf & bootstrap
